Dissertation SQL Database and Front End

For my dissertation, I created two deliverables, a SQL and NoSQL database with front ends to use and test them. A comparison was made between them to analyse retreival time, perceived retreival time, scalability, and accuracy. The goal of this was to conclude which solution performed better while containing data related to healthcare.

This is the SQL deliverable hosted on Microsoft Azure. The front end is an Azure static web app connected to the database using the GraphQL and Rest API. The files for this project can be found HERE or the images, with an included overview, can be found below.

Business Intelligence Tool

For my HND, I had to create a tool for the business intelligence unit. I decided to create a tool which provides the user with some information about their Destiny 2 character's level, specifically, the power levels of their gear. This tool would work out their actual power level, how close they are to the next level, and their lowest leveled piece of gear. This information was inputted manually which could built upon by using a connection to their API, however, it was not needed for this unit. Images of the program can be found below.
