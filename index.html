My name is Connor Walton, I have recently graduated from my Bachelors degree in Applied Computing. Throughout my academic years, I have developed projects which are displayed on this page, below.

Dissertation SQL Database and Front End

For my dissertation, I created two deliverables, a SQL and NoSQL database with front ends to use and test them. A comparison was made between them to analyse retreival time, perceived retreival time, scalability, and accuracy. The goal of this was to conclude which solution performed better while containing data related to healthcare.

This is the SQL deliverable hosted on Microsoft Azure. The front end is an Azure static web app connected to the database using GraphQL. The files for this project can be found HERE or the images, with an included overview, can be found below.
